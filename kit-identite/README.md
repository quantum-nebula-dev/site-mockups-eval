# Périclès Gestion — Kit d'identité visuelle

Concept : **la colonne de pilotage**. Une structure verticale tient cinq tracés, représentant les cinq domaines de gestion réunis sous un point de responsabilité unique.

## Logos

- `logos/pericles-logo-horizontal.svg` : version principale sur fond clair.
- `logos/pericles-logo-horizontal-inverse.svg` : version principale sur fond sombre.
- `logos/pericles-logo-compact.svg` : version centrée pour formats carrés ou verticaux.
- `logos/pericles-symbole.svg` : favicon, avatar et petits espaces.
- `logos/pericles-symbole-inverse.svg` : symbole sur fond sombre.

## Couleurs

- Forêt : `#1E332B`
- Calcaire : `#F3F0E8`
- Bronze : `#A48252`
- Pierre : `#716D64`
- Sauge : `#AAB9B0`
- Blanc : `#FFFFFF`

Les variables prêtes pour le web sont dans `tokens.css`.

## Typographies

- Newsreader : titres, citations et parole institutionnelle.
- Instrument Sans : textes, interfaces, tableaux et informations.

## Ressources

- `patterns/pericles-traces.svg` : motif principal.
- `icons/` : cinq pictogrammes de domaines.
- `templates/` : carte de visite, couverture de rapport et publication LinkedIn.
- `Charte-identite-visuelle-Pericles-Gestion.pdf` : guide complet.

## Règles essentielles

1. Ne jamais modifier le nombre ou l'ordre des cinq tracés dans le symbole.
2. Utiliser le logo horizontal par défaut.
3. Ne pas employer de dégradé, d'ombre ou d'effet métallique.
4. Réserver le bronze aux accents et repères.
5. Conserver un contraste suffisant pour tous les textes.
